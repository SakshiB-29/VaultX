# Bank Management System (C++ + MySQL)

## 💡 Overview
A simple console-based banking system built in C++ using Object-Oriented Programming and MySQL. Users can create accounts, deposit/withdraw money, and check balances.

## 🧠 Tech Used
- C++ (OOP concepts)
- MySQL (Database)
- MySQL Connector/C++

## ⚙️ Features
- Account creation
- Deposit / Withdraw
- Balance inquiry
- Transaction logging
